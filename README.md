# 社内お問い合わせナビ

社内の問い合わせ対応を減らし、  
社員の自己解決を促す **社内向けヘルプナビツール**です。

問い合わせ前に確認すべき内容を案内し、  
解決しなければそのまま問い合わせへ誘導します。

## できること

### 社員向け

- よくある「困りごと」をカテゴリから選択
- チェックリスト形式で確認手順を案内
- 解決しなければ、そのまま問い合わせ可能
- 問い合わせ時に端末名・画像添付が可能

### 管理者向け（情シス・管理部門）

- カテゴリ・事象（ナレッジ）の管理
- 部署ごとの問い合わせ通知先設定
- 問い合わせ件数・閲覧数の集計
- CSV出力による問い合わせ分析

## 特徴

- **閉域ネットワーク前提**
- **社内サーバー設置型**
- 社員側はログイン不要
- 管理者のみID・パスワード管理
- DBはSQLite（軽量・運用が簡単）
- SMTP不要（フォーム送信URL対応）

---

- スタートアップ
- 中小企業
- 情シス・管理部門が少人数の現場
- 情シス専任がいない組織

## 導入方法（概要）

1. 社内サーバーにファイルを配置
2. Node.js（18以上）をインストール
3. start.bat を実行
4. ブラウザで社内IPにアクセス

※ 詳細な手順は導入時に案内します
